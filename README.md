# Codeforces Dark Mode Pro 🚀

I built this because I was tired of staring at blinding white screens during late-night Codeforces grinds. Most dark mode extensions out there just slap a generic invert filter on everything and call it a day, which completely ruins the math formulas and messes up the native feel. 

I wanted something that feels premium, native, and actually reduces eye strain. So, I completely rebuilt the CSS layer.

### What it does:
- **Pure Zinc Dark Theme**: Uses a carefully crafted soft dark palette (Zinc 900/800) instead of harsh pure black to reduce eye strain.
- **Flawless MathJax**: Every single math formula renders perfectly in bright, readable text.
- **Custom Rating Colors**: I mapped every Codeforces rating color (from Newbie to Legendary Grandmaster) to a specific dark-mode equivalent so they pop perfectly without hurting your eyes.
- **Smart Image Inversion**: Banners, logos, and empty avatars blend seamlessly into the background instead of looking like giant white boxes.

### How to Install (Unpacked)
You can load this directly into your browser in a few clicks:
1. Download this repository as a `.zip` file using GitHub's green **"Code" -> "Download ZIP"** button.
2. Unzip the folder anywhere on your computer.
3. Open Chrome/Edge/Brave and go to `chrome://extensions/`.
4. Turn on **Developer mode** in the top right corner.
5. Click **"Load unpacked"** and select the unzipped folder.

That's it. Enjoy the dark mode and happy coding!
